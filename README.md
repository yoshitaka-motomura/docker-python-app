# Pyhon API Service for Flask at Docker Example

## Description
This is a simple example of a Python API service for Flask at Docker.

## Requirements
- Docker
- Docker Compose
- Python 3.11
- Flask 3.0.x

## Commands

**Dev mode**
running on http://localhost:5000
```shell
flask --app src/app run
```
